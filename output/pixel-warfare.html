Pixel Warfare

Pixel Warfare is a superb multiplayer FPS game featuring blocky graphics and 8-bit music. The gameplay is classic, fast-paced, and tactical, featuring various weapons and maps. Choose a game from the menu screen to get started!

How to Play

Start an online battle instantly

Pixel Warfare features active game rooms to join and instantly start fighting in online battles! There are several maps and various settings in each game room, so you can find the perfect game.

Use your arsenal of weaponry

Pixel Warfare features a variety of weapons you can scroll through when you’ve started a match. There’s a shotgun, sniper, rocket launcher, machinegun, and many other weapons. They all contain a set amount of ammunition. You have every weapon by default, so there is no need to unlock them.

Create a private room

If you want full control over the game, you can start a personalized game room with custom options. Here you can choose the game mode, map, round duration, max players, weapons to allow, and more.

Build up your stats

Pixel Warfare will remember your kills, deaths, and K/D ratio, displaying them to you in the menu screen, so you can see how well you’re doing over time.
